# Bob's 27 Darts Calculator

A Progressive Web App (PWA) for tracking scores in the Bob's 27 darts game.

## Features

- ✅ Track your score through all 21 rounds (Double 1-20 + Bullseye)
- ✅ Undo last round
- ✅ Keyboard shortcuts (0-3 for scoring, Backspace/U to undo)
- ✅ **Installable on desktop, phone, and tablet**
- ✅ **Works offline**
- ✅ Mobile-optimized full-screen design

## Installation

### On Desktop (Chrome, Edge, Brave)
1. Visit the app URL
2. Look for the install icon (⊕) in the address bar
3. Click "Install" to add it to your desktop

### On iPhone/iPad
1. Open in Safari
2. Tap the Share button
3. Select "Add to Home Screen"
4. Tap "Add"

### On Android
1. Open in Chrome
2. Tap the menu (⋮)
3. Select "Install app" or "Add to Home Screen"

## GitHub Pages Setup

To host this app on GitHub Pages:

1. Go to repository settings: https://github.com/dowdarts180-lgtm/bobs-27-darts/settings/pages
2. Under "Build and deployment":
   - Source: **Deploy from a branch**
   - Branch: **main** and folder **/ (root)**
   - Click **Save**

3. Upload the icon files (`icon-192.png` and `icon-512.png`) to the repository root

Your app will be available at: **https://dowdarts180-lgtm.github.io/bobs-27-darts/**

## How to Play

Bob's 27 is a darts game where you:
- Start with 27 points
- Progress through doubles (D1-D20) and finish with Bullseye (D25)
- Each hit adds points (2× the number)
- Each miss subtracts points (2× the number)
- Goal: Complete all 21 rounds with the highest score
